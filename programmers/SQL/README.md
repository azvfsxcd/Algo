### 프로그래머스 SQL 코딩테스트 연습

---

SELECT 1 - 모든 레코드 조회하기

```SQL
SELECT *
FROM ANIMAL_INS
ORDER BY ANIMAL_ID;
```

SELECT 2 - 역순 정렬하기

```SQL
SELECT NAME, DATETIME
FROM ANIMAL_INS
ORDER BY ANIMAL_ID DESC;
```

SELECT 3 - 아픈 동물 찾기

```SQL
SELECT ANIMAL_ID, NAME
FROM ANIMAL_INS
WHERE INTAKE_CONDITION = "Sick"
ORDER BY ANIMAL_ID;
```

SELECT 4 - 어린 동물 찾기

```SQL
SELECT ANIMAL_ID, NAME
FROM ANIMAL_INS
WHERE INTAKE_CONDITION != "Aged"
ORDER BY ANIMAL_ID;
```
